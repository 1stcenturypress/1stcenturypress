Selecting Multiple DOM Nodes

We start out with 4 identical DIVs with the same CLASS assigned (.this-class).

This is a class 1
This is a class 2
This is a class 3
This is a class 4

In JS, we create a new variable:

const elems = document.querySelector(".this-class");
console.log(elems.innerText);

This will initially only return the first DIV: 'This is a class 1', not all 4 DIVs.

If we change the JS to read:

const elems = document.querySelectorAll(".this-class");
console.log(elems);

We get a NodeList containing each DIV as an array item:

NodeList(4) [div.this-class, div.this-class, div.this-class, div.this-class]

What we can do with this, we will dive into later.

Another Example

Let's build an ordered list which resembles this...

1. This is One
2. This is Two
3. This is Three

Then in JS code this...

const elements = document.querySelectorAll("li");
elements.forEach((node) => { node.innerText = "This is CHANGED!"; });

We query for ALL the LI elements on the page, and loop through them with a forEach, changing the innerText as we find them. And we end up with 3 changed list items...

  1. This is One
  2. This is Two
  3. This is Three
