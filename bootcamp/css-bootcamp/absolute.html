To demonstrate absolute positioning, we will need to first create a parent box set to relative positioning. Absolute positioning only works when incorporated with a relative-positioned element.

We start with our "parent" box set to RELATIVE and add a child box inside

Child 1: Top Right

Without having set the parent box to relative, the child box would appear at the top right of the ENTIRE PAGE rather than in the defined structure of the parent box!

SO, ... an absolute position must be contained within a relative position in order to give it an "environment" to be absolute in.
