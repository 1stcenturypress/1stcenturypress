VIEWPORT Meta Element

VSCode will automatically insert the necessary head, title, body, and meta tags needs for a clean start for a page -- at least when you use the shortcuts for this:
HTML:5 + ENTER or ! + ENTER.

However, if you are manually adding these basic structures, its the META tag content that we are focusing on here and you must be certain to include in all your pages.

Instruct the browser that this page will be responsive to the various device screen types by adding: META NAME="VIEWPORT" CONTENT="WIDTH=DEVICE-WIDTH, INITIAL-SCALE=1.0"

[Note: the META CHARSET='UTF-8" is what instructs the browser to use the latest character sets, including emojis.]

Responsive Embeds

Auto-adjusting a video or image to fit proportionately within your page on any device can require some CSS tweaking.

We create a .CONTAINER class {} with POSITION: RELATIVE, OVERFLOW: HIDDEN, with WIDTH:100% and PADDING-TOP:56.25% (which is 9 / 16 * 100 for 16x9 ratio).

Then assign .CONTAINER IFRAME {} to have POSITION: ABSOLUTE with TOP: ,RIGHT: ,BOTTOM: and LEFT: set at 0 and WIDTH and HEIGHT set to 100%.

Inside the DIV we call the 'container' class and insert the IFRAME content from the video.

And you should be seeing the video fill 100% width on any device at any resolution.
