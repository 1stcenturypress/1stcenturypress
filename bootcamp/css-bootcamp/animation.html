Animated Elements

CSS can create motion without extensive programming needed.

Notice the orange circle moving left to right across the top margin of this page (and then returns). It was created by 2 specific sets of commands:
(1) additional properties in a CSS box definition, such as ANIMATION-NAME:, ANIMATION-DURATION:, ANIMATION-ITERATION-COUNT:, and ANIMATION-DIRECTION:
and (2) a separate CSS definition called @KEYFRAMES, which calls an animation function (in this case named 'slideMe') which assigns FROM { } and TO { } properties.

(And box-shadow has been applied as well for a floating effect.)

For a second example, we have created another circle, this one red, that slides from right to left but also shifts colors as it crosses the screen.

And for added effect, we've dropped the element down one layer (using Z-MARGIN:) to appear as if behind the body text. The H1 has been given a text-shadow as well to add to the illusion.
