CSS 201 - GRID

As great as FLEXBOX is over TABLE, GRID takes the layout game up a notch from Flexbox.

NOTE: We have added BOX-SIZING: BORDER-BOX; to the page STYLE to best use the layout features of these advanced CSS formats.

While Flexbox is great for vertically aligned layouts (if you think about it its mostly a stacked format), Grid is designed for a template layout.

COLUMNS, ROWS, and GUTTERS (GAPS)

We start with a container element called #grid containing a 3 x 3 grid (9 boxes originally)

Let's break down the process of creating the above grid layout: Item1 and Item7 are both a GRID-COLUMN-START:1 and GRID-COLUMN-END:4. This means Item1 "consumes" the space of columns 1 through 3 of the 1st row, Item7 consumes columns 1 to 3 of the 4th row.

Item4 and Item5 both consume columns 2 through 3 of the 2nd and 3rd row respectively, using GRID-COLUMN-START:2 and GRID-COLUMN-END:4.

And Item6 consumes the rows 2 through 3 of the 1st column using GRID-ROW-START:2 and GRID-ROW-END:4.
