CSS 201 - STICKY Positioning

Return To CSS 201 Main

Sticky Positioning is very similar to Fixed Positioning, except that it "sticks" only to its parent area. Meaning, it can scroll away when other areas in a page dominate/control the screen space.

And you can have multiple occurrences of a Sticky Box in a single page.
