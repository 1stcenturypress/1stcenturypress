Fixed Positioning is when an element is set to a particular coordinate on the visible screen regardless of the other page content.

For this example, we placed the box at the top of the page to simulate a HEADER. And the BODY has been given a min height of 1500px to ensure scrolling. The box will appear "fixed" to the top of the screen.
