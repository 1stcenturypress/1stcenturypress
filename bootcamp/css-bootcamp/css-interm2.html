PSEUDO ELEMENTS

Pseudo Elements are similar to Pseudo Selectors/Classes. With Pseudo Elements, CSS steps into the world of HTML coding. Again, we use our boxes below to demonstrate.

::BEFORE and ::AFTER

Behaves like inline HTML.

The BEFORE gets assigned red border, AFTER gets the yellow border, and the default box gets the solid blue. We have shifted the BEFORE to -30px for TOP/LEFT and the AFTER to -30px BOTTOM/RIGHT. The layer effect comes from applying a Z-INDEX of -1 to BEFORE and 1 to AFTER. (This leaves the default box at 0 or between them).

REAL WORLD Example

Say we want to apply an effect to a hyperlink, like in the following text. We want a border that impersonates an underline for the link and then to highlight the link when hovered over.

Lorem ipsum dolor sit amet consectetur, adipisicing elit. I have a border (hover over me for a highlight). Perspiciatis animi aliquam, qui quasi porro velit non eveniet, ipsam cupiditate ipsa ea magnam facilis pariatur tenetur repudiandae magni consequuntur repellat repellendus!

This is accomplished through CSS assignment of conditions to the A HREF and the HOVER elements, through RELATIVE/ABSOLUTE positioning, and use of the Z-INDEX for layering.

Targeting First Letter / First Line Of Elements

Say we want to enlarge the first letter of a paragraph/article/post, etc. You just create a style element for the block of text using ::first-letter and give it specified font atrributes like size or weight and float it to the left to allow for multi-line wrap. Like we just did for this paragraph.

Or perhaps we want to change the attributes of the first line of a block of text, such as this one. We can simply create a style element for the block using ::first-line. USER SELECTION Customization

You can control the formatting of a user-selected text on your page. This is when a user "highlights" a portion of the text on the page for temporary focus. This is simply done in CSS with ::SELECTION (does not require a host element) and the attributes you want changed. Go ahead. Select some text in this paragraph (or anywhere on this page) to test it out. I have set the selection to aquamarine background with black text.
